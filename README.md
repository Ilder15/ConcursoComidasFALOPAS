
# 🍽️ Concurso de Comida FALOPAS

Aplicación web para gestionar concursos de cocina familiares.

## Características
- Registro de participantes (cocineros e invitados)
- Alta de platos
- Sistema de calificación (1-10 estrellas)
- Bloqueo de auto-calificación
- Ranking automático con promedios
- Persistencia en localStorage
- Diseño responsive para móviles

## Uso local
1. Clonar el repositorio
2. Abrir `index.html` en el navegador
